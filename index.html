1. Terdapat satu masukan parameter untuk suatu program yang berupa bilangan bulat positif. Tugas untuk soal ini adalah membuat suatu program dimana masukan parameter bilangan bulat positif akan menghasilkan suatu pola. Misal masukan parameter dilambangkan dengan r.

2. Terdapat suatu perlombaan memindahkan bola dari titik A ke titik B dengan batas jumlah waktu tertentu. Pemain yang berhasil memindahkan bola dengan jumlah terbanyak akan menempatkan peringkat tertinggi. Dimana peringkat pemain menerapkan sistem Dense Ranking, artinya jika terdapat dua atau lebih pemain memiliki jumlah bola yang berhasil dipindahkan dengan nilai sama, maka pemain tersebut memiliki peringkat yang sama. Dalam perlombaan ini sudah terdapat para pemain dengan jumlah bola tertinggi dan terdapat pemain penantang untuk mengalahkan pemain dengan peringkat tertinggi saat ini. Buatlah suatu program untuk menentukan peringkat pemain penantang. Program menerima masukan dua parameter berupa array dari jumlah bola yang berhasil dipindahkan dari titik A ke titik B, parameter pertama adalah jumlah bola untuk peringkat tertinggi saat ini dan parameter kedua adalah jumlah bola pemain penantang.
